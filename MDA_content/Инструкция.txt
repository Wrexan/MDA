Функции приложения:
 + Данное приложение позволяет по названию модели устройства одновременно искать данные в *.xml прайсе и интернет базе.

Особенности:
 + Убирает необходимость искать одно и то же в прайсе и в браузере.
 + Убирает необходимость выбирать лист прайса и искать модель визуально.
 + Быстрый поиск почти не трогая мышь. Переключаешься на приложение -> вводишь модель -> жмешь энтер -> результат.
 - Проводить квитанции надо через сайт, но это нужно реже.
 - Некоторые модели (устаревшие) не обрабатываются, или обрабатываются криво. Не удалял из прайса.
 - Не поддерживается Windows XP и ниже :(

Использование:
Прайс должен иметь формат *Прийс*.xml и находиться на рабочем столе.
При первом запуске зайдите в настройки приложения и введите ваши Логин и Пароль от интернет базы.
При переключении на приложение можно сразу вводить модель, выделять посковую строку не обязательно.

Три строки вверху - кеш для прайса.
    Нужны для просмотра длинных строк, для заказа (можно копировать строку по двойному клику),
    и в ситуации, когда надо найти совместимый разъем от других моделей.
Левее отображается текущий прайс и статус соединения с базой.
В процессе ввода текста производится поиск в прайсе и найденные модели выводятся в виде списка а фирмы над строкой поиска.
	Стрелки вверх/вниз - выбор модели из списка.
	Стрелки влево/вправо - выбор фирмы. Обновляет список моделей.
	Enter/Click - вывод данных прайса/базы.
	    При этом могут появиться кнопки для поиска в интернет базе совместимых моделей, например A525 и A526
	    (В прайсе данные модели указаны через рапятую).
	    При обнарушении ссылки на другую модель (в прайсе: "см Redmi 6 Pro") происходит редирект на эту модель. Изначальная модель выводится в виде кнопки.
	Alt - переключение таблиц Запчасти/Аксессуары.
	Shift - выделяет всю строку поиска (для быстрого стирания).
	    Поиск не зависит от регистра, не нужно вводить с зажатым Shift.
	При наведении на поле таблицы появляется подсказка с полным ее содержимым.
	При двойном клике по строке таблицы Запчасти/Аксессуары часть полей копируется и подчеркивается.
	    Скопированную строку можно использовать для заказа.
	При клике на строку прайса строка помещается в кеш.
Под строкой поиска находится полоса загрузки с сайта, если заполнена - нет соединения с сервером.
--Если введенный запрос не обнаружен в прайсе как модель, поиск идет только в интернет базе. Например "loca".

Следующие опции помогают при поиске, потому включены по умолчанию.
    "Искомая модель" - фильтрует модели веб таблицы, если чего-то не находите, попробуйте отключить.

    "Модель прайса" - поиск в веб строго по модели из прайса(что появляются в виде кнопок). Если отключено - поиск модели в базе по тексту из строки поиска.
	"Латиница" - автозамена русских/украинских символов на латинские.
	"Минимум 2" - поиск начиная с 2х символов. Отключите, если надо найти один символ (например кириллическую "с").
	"Разширенный поиск" - не реализовано. Поиск по всем 4м полям.

т.к. веб база и Excel по разному относятся к кириллице, в прайсе поиск происходит как есть, для веб некоторые символы заменяются (с6502). По началу желательно переповерять.

В настройках можно изменить шрифт под свой монитор.

Программа улучшается в свободное время.
