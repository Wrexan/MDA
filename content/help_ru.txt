Функции приложения:
 * Данное приложение позволяет по названию модели устройства одновременно искать данные в *.xml прайсе и интернет базе.
 * Приложение тестировалась на Win 7 и Win 10

Использование:
    Прайс должен иметь формат *Прайс*.xml и находиться на рабочем столе или рядом с исполняемым файлом.
    При первом запуске введите ваши Логин и Пароль от веб базы.
    При переключении на приложение можно сразу вводить модель, выделять поисковую строку необязательно.

Интерфейс:
    Три строки вверху - кеш для прайса.
        Нужны для просмотра длинных строк и копирования нужного поля.
    Левее отображается текущий прайс и статус соединения с базой.
    В процессе ввода текста производится поиск в прайсе, и найденные модели выводятся в виде списка, а фирмы над строкой поиска.
        Стрелки вверх/вниз - выбор модели из списка.
        Стрелки влево/вправо или колесо мышки(курсор в верхней части или открыт список моделей) - выбор фирмы. Обновляет список моделей.
        Enter или Клик по модели - вывод данных прайса/базы.
            При этом могут появиться кнопки для поиска в интернет базе совместимых моделей, например A525 и A526
            (В прайсе данные модели указаны через запятую).
            При обнаружении ссылки на другую модель (в прайсе: "см Redmi 6 Pro") происходит редирект на эту модель. Изначальная модель выводится в виде кнопки.
        Enter/Двойной клик по строке ввода когда нет списка моделей - проявляет список моделей.
        Alt - переключение таблиц Запчасти/Аксессуары.
        Shift - выделяет всю строку поиска (для быстрого стирания), а так же переводит курсор на правое поле (из которого не отправляется статистика) и обратно.
            Поиск не зависит от регистра, не нужно вводить с зажатым Shift.
        При наведении на поле таблицы появляется подсказка с полным ее содержимым.
        При двойном клике по строке таблицы часть полей копируется и подчеркивается.
            Скопированную строку можно использовать для заказа.
        При клике на строку прайса, строка помещается в кеш.
    По правому краю списка расположены небольшие пустые поля. При их выборе будет производиться поиск модели в строке, но без сохранения в статистику (если запрос делается для себя, а не для клиента).
    Под строкой поиска находится полоса загрузки с сайта, если заполнена - нет соединения с сервером.

Следующие опции помогают при поиске, некоторые включены по умолчанию (можно включить\отключить в настройках).
    "Фильтр модели" - фильтрует модели веб таблицы. Может помочь при большом кол-ве результатов. При поиске всегда сбрасывается на "отключено".
    "Показать дату" - разворачивает колонку с датой поступления товара на склад.
    "Латиница" - автозамена русских/украинских символов на латинские.
    "Два символа" - поиск начиная с 2х символов. Отключите, если надо найти один символ (например кириллическую "с").
    "Расширенный поиск" - Поиск по всем 4м полям.

В процессе пользования программа отправляет на сервер статистику.
    В момент отправки запроса к интернет базе содержание запроса сохраняется для отложенной отправки.
    Собранные с клиентских программ данные можно запросить с сервера нажав на кнопку "Статистика".
    Окно статистики содержит несколько графиков информирующих о потребностях клиентов и о тенденциях.

Решение проблем:
    Если что-то не находится - снимаем галочку с "Модель прайса"
    Если мелкий шрифт - в настройках можно изменить под свой монитор.
    По вылетам и ошибкам пишем мне (https://t.me/wrexan).

Что может не работать:
    Поиск не по модели, например "стекло" - используйте расширенный поиск.
    Не используются:
      Не актуальные листы: fly alcatel blackberry htc.
      Старые модели Nokia и прочие.
      Листы со скидками, восстановлением информации и прочим.
    Т.к. веб база и Excel по-разному относятся к кириллице, в прайсе поиск происходит как есть, для веб некоторые символы заменяются (в прайсе кириллицей: с6502).

Структура прайса:
_______________1_____________|___________2_________|__________6____________|___________7____________|______8_______
___название модели, назв...__|_____________________|_название осн. модели__|________________________|______________
_____________________________|__виды работ/запчасть|_________цена__________|_совм. запчасть или ..._|__примечание__

Первая строка: считается той, в которой не пуста первая ячейка.
    Столбец 1: Только названия моделей. Сначала основная модель, через запятую другие названия. Поиск запчасти идет по первому названию. В скобках можно указать информацию, не входящую в модели (NFC, 5G)
    Столбец 6: Если напротив названия в 6м столбце прописана модель ("см Redmi 6 Pro"), происходит редирект на эту модель.
Следующие строки: все строки с первой пустой ячейкой
    Столбец 2: Виды работ
    Столбец 6: Цена
    Столбец 7: Совместимая деталь: "Запчасть" "Производитель" "Модель" ("Примечание") = (Стекло Xiaomi Mi11 lite (вк)). Примечание может быть любым или отсутствовать. Можно добавлять несколько запчастей раздкляя их с помощью "или".
    Столбец 8: Примечание

Прочие ячейки не обрабатываются
